# Arq BIM

Visualizador 3D moderno para **arquitetura e BIM** — suporte a **GLB, GLTF, OBJ e FBX**.

---

## Cores do Projeto
- **Rosa Claro**: #e07ab0
- **Azul Claro**: #4a9eff

---

## Como Usar

1. Exporte seus modelos em GLB (recomendado com Draco) ou GLTF
2. Acesse o painel admin em `/admin`
3. Faça upload dos arquivos
4. Compartilhe o link do viewer

## Rodando Localmente

```bash
cp .env.example .env
npm install
npm run dev
```

Acesse: http://localhost:3000/admin
